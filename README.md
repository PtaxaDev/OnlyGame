# Запуск проекта через VS Code Live Server

Этот проект — простой статический сайт на HTML, CSS и JS. Ниже описано, как быстро его запустить с помощью **Live Server** в Visual Studio Code.

## Шаг 1. Установка VS Code

Если у вас ещё не установлен [Visual Studio Code](https://code.visualstudio.com/), скачайте и установите его для вашей ОС.

## Шаг 2. Установка Live Server

1. Откройте VS Code.
2. Перейдите в **Extensions** (Расширения) с помощью боковой панели или комбинации `Ctrl+Shift+X`.
3. В поиске введите `Live Server`.
4. Найдите расширение от **Ritwick Dey** и нажмите **Install**.

## Шаг 3. Клонирование проекта

Откройте терминал и склонируйте проект с GitHub:

```bash
git clone https://github.com/ВАШ_АККАУНТ/ВАШ_РЕПО.git
```

Перейдите в папку проекта:

```bash
cd ВАШ_РЕПО
```

## Шаг 4. Открытие проекта в VS Code

1. В VS Code нажмите **File → Open Folder** и выберите папку проекта.
2. Дождитесь, пока VS Code загрузит все файлы.

## Шаг 5. Запуск Live Server

Есть несколько способов:

* **Способ 1:**
  Нажмите правой кнопкой мыши на `index.html` → **Open with Live Server**.

* **Способ 2:**
  В правом нижнем углу VS Code нажмите **Go Live**.

После этого ваш сайт откроется в браузере по адресу, обычно `http://127.0.0.1:5500`.

## Шаг 6. Просмотр изменений

* Любые изменения в HTML, CSS или JS автоматически будут обновляться в браузере благодаря Live Server.
* Для остановки сервера нажмите кнопку **Port: 5500 → Stop Live Server** в правом нижнем углу VS Code.

## Дополнительно

* Если сайт использует изображения или другие ресурсы, убедитесь, что пути к ним правильные.
* Если что-то не работает, проверьте консоль браузера (`F12`) на наличие ошибок.
